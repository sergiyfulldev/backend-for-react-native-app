# (Admin) Users

Endpoint to retrieve a list of all the users profiles.
Specifically consumed via the dashboard.

**URL** : `/api/admin/users/`

**Method** : `GET`

**Auth required** : YES

## Response

**Code** : `200 OK`

**Content example**

```json
{
  "status": "success",
  "message": "",
  "users": [/*...*/]
}
```
